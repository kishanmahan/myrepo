Matlab:-
“Certification ka design kaise hoga” ye define karta hai

🔥 Simple Example:-
Tu bolta hai:-
Kaunse users ka review hoga
Kaun reviewer hoga
Kaunsa access include hoga

()🔹 🔥 Inclusion Rule kya hai?
👉 Inclusion Rule = Kaun include hoga certification me
Matlab:-
“Kin users / accounts / access ko certification me lana hai”

🔸 Example:-
definition.setInclusionRule("OnlyActiveUsers");

👉 Rule logic:-
if(user.isActive()){
   return true; // include karo
}
✅ Result:-
Sirf active users certification me aayenge.
----------------------------------------------------------------------------
🔹 🔥 Exclusion Rule kya hai?
👉 Exclusion Rule = Kisko hataana hai (filter out)
Matlab:-
“Jo include ho chuke hain unme se kisko remove karna hai”

🔸 Example:-
definition.setExclusionRule("ExcludeAdmins");

👉 Rule logic:-
if(user.role == "Admin"){
   return true; // exclude karo
}
✅ Result:-
Admin users ❌ remove ho jayenge
Baaki sab rahenge.
----------------------------------------------------------------------------
