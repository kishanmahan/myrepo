🔥 PolicyViolation kya hai?
👉 PolicyViolation = jab koi user company ki security policy tod deta hai

➡️ Matlab:-
User ke paas aisa access aa gaya jo allowed nahi hai

💡 Simple Example:-
👉 Company rule:-
Developer + Production Access ❌ (allowed nahi)

👉 User ke paas:-
Developer role + Production access
➡️ 🔥 Ye ho gaya Policy Violation.

🔥 SailPoint me PolicyViolation kya karta hai?
👉 Ye ek object hota hai jo:-
violation detect karta hai
violation ka detail store karta hai
report / audit me show hota hai

🔥 PolicyViolation me kya kya hota hai?
👉 Typical fields:-
user (Identity)
violated policy name
risk level (High / Medium / Low)
description
conflicting access

()🔥 1️⃣ Resolution (kya hota hai?)
👉 Resolution ka matlab hai problem ko poori tarah solve kar dena (final fix)

Root cause ko fix karta hai

📌 Example:-

🎯 Result:-

👉 Mitigation ka matlab hai risk ko temporarily control ya reduce karna.

✅ Kya karta hai:-
Problem ko turant control karta hai
Lekin permanent fix nahi hota

📌 Example:-
User ke paas risky access hai (remove nahi kar sakte immediately)
👉 Mitigation:-
Monitoring enable kar di
Approval mandatory kar diya
Time-based access laga diya

🎯 Result:-
✔ Risk kam ho gaya
❌ Problem fully solve nahi hui
