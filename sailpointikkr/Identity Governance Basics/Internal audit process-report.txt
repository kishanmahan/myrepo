()Internal audit process:-
1️⃣ Internal Audit Process kya hai?
Internal Audit = Company khud apne system ko check karti hai.
Goal:-
Security rules follow ho rahe hain ya nahi
Unauthorized access hai ya nahi
Policy violations detect karna

Example:-
User : Rahul
Role : FinanceAdmin + DatabaseDelete
Result : Policy Violation

2️⃣ Internal Audit ke Steps:-
1️⃣ Data Collection:-
System se data collect hota hai
Example:-
Users
Roles
Access
Logs

2️⃣ Policy Check:-
Security rules apply kiye jate hain.
Example rule:-
Finance role ko DatabaseDelete access nahi milna chahiye

3️⃣ Violation Detection:-
Agar rule break hua → Violation
Example:-
User : Rahul
Role : FinanceAdmin
Access : DatabaseDelete
Result : Violation

4️⃣ Audit Report Generate:-
User      Role           Status
--------------------------------
Rahul     FinanceAdmin   Violation
Amit      Employee       Compliant
Neha      HR             Compliant

Internal Audit Process (Context Based Project Code)
import sailpoint.api.SailPointContext;
import sailpoint.object.Identity;
import sailpoint.object.QueryOptions;
import sailpoint.object.Filter;
import sailpoint.tools.GeneralException;

import java.util.Iterator;

public class InternalAuditProcess {

    private SailPointContext context;

    public InternalAuditProcess(SailPointContext context) {

        this.context = context;

    }

    public void runAudit() throws GeneralException {

        QueryOptions qo = new QueryOptions();

        Iterator<Identity> identities = context.search(Identity.class, qo);

        System.out.println("===== INTERNAL AUDIT REPORT =====");

        while (identities.hasNext()) {

            Identity id = identities.next();

            String name = id.getName();
            boolean violation = false;

            // Example Policy Check
            if (id.getAssignedRoles() != null && id.getAssignedRoles().toString().contains("FinanceAdmin")
                    && id.getAssignedRoles().toString().contains("DatabaseDelete")) {

                violation = true;

            }

            System.out.println("User : " + name);

            if (violation) {

                System.out.println("Audit Result : POLICY VIOLATION");

            } else {

                System.out.println("Audit Result : COMPLIANT");

            }

            System.out.println("------------------------------");

        }

    }

}
Example: runAudit() ko kaun call karega:-
import sailpoint.api.SailPointContext;
import sailpoint.tools.GeneralException;

public class AuditMain {

    public static void main(String[] args) throws GeneralException {

        SailPointContext context; 

        InternalAuditProcess audit = new InternalAuditProcess(context);

        audit.runAudit();   // yaha call ho raha hai

    }

}
