🔹 Comparator kya hai?
Comparator ek interface hai Java me jo do objects ko compare karne ka logic define karta hai.
Ye decide karta hai ki kaunsa object pehle aayega aur kaunsa baad me.
Methods: compare(T o1, T o2)
Return value:-
<0 → toh o1 pehle return
0 →  equal sath me
>0 → toh o2 pehle return hoga.

package com.example.sortexample;

import java.util.List;
import java.util.ArrayList;
import java.util.Collections;
import java.util.Comparator;

// Identity class
class Identity {
    private String name;

    public Identity(String name) {
        this.name = name;
    }

    public String getName() {
        return name;
    }

    @Override
    public String toString() {
        return name;
    }
}

// Custom Comparator class
class IdentityNameComparator implements Comparator<Identity> {
    @Override
    public int compare(Identity i1, Identity i2) {
        return i1.getName().compareTo(i2.getName()); // Ascending order
    }
}

public class SortExample {
    public static void main(String[] args) {
        List<Identity> identities = new ArrayList<Identity>();
        identities.add(new Identity("Rahul"));
        identities.add(new Identity("Ram"));
        identities.add(new Identity("Shyam"));

        System.out.println("Before Sorting: " + identities);

        // Sort using custom comparator
        Collections.sort(identities, new IdentityNameComparator());

        System.out.println("After Sorting Ascending: " + identities);
    }
}

✅ Output:

Before Sorting: [Rahul, Ram, Shyam]
After Sorting Ascending: [Ram, Rahul, Shyam]
🔹 Descending Order

Agar descending chahiye to comparator me reverse kar do:

class IdentityNameDescendingComparator implements Comparator<Identity> {
    @Override
    public int compare(Identity i1, Identity i2) {
        return i2.getName().compareTo(i1.getName()); // Descending
    }
}

// Use:
Collections.sort(identities, new IdentityNameDescendingComparator());

✅ Output:

[Shyam, Rahul, Ram]
