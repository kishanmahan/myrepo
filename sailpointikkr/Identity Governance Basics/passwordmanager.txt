()🔹PasswordManager kya hai?
👉 PasswordManager ek service/utility component hai jo:-
👉 Password ko manage karta hai:-
set
reset

change

🔥 🔹 Ye karta kya hai?
1️⃣ Password set / change:-
👉 User ka password update karta hai
passwordManager.changePassword(context, identity, "Rahul@123");

2️⃣ Password reset:-
👉 Admin ya forgot password flow
passwordManager.resetPassword(context, identity);

3️⃣ Password validate:-
👉 PasswordPolicy ke against check karta hai
passwordManager.validatePassword(context, identity, "Rahul@123");

4️⃣ Password hash / encryption:-
👉 Password ko secure format me convert karta hai
❌ Plain text store nahi hota
✔ Hash form me store hota hai

5️⃣ Password history maintain:-
👉 Purane passwords track karta hai
👉 Repeat password block karta hai

6️⃣ Account lock / unlock support:-
👉 Failed attempts ke baad account lock

7️⃣ Password expiration handle:-
👉 Expired password detect karta hai.

import sailpoint.api.SailPointContext;
import sailpoint.api.SailPointContextFactory;
import sailpoint.api.PasswordManager;
import sailpoint.object.Identity;
import sailpoint.object.PasswordPolicy;

import java.util.List;
public class PasswordManagerFullDemo {

    public static void main(String[] args) {

        try {

            // 1️⃣ Context create
            SailPointContextFactory factory = new SailPointContextFactory();
            SailPointContext context = factory.getCurrentContext();

            // 2️⃣ Identity fetch
            Identity identity = context.getObjectByName(Identity.class, "rahul.user");

            // 3️⃣ PasswordManager create
            PasswordManager pm = new PasswordManager(context);

            // -------------------------------
            // 🔐 SET PASSWORD
            // -------------------------------
            pm.setPassword(identity, "Rahul@123");
            System.out.println("Password set successfully");

            // -------------------------------
            // 🔄 CHANGE PASSWORD
            // -------------------------------
            pm.changePassword(identity, "Rahul@123", "Rahul@456");
            System.out.println("Password changed successfully");

            // -------------------------------
            // 🔁 RESET PASSWORD
            // -------------------------------
            String tempPass = pm.resetPassword(identity);
            System.out.println("Temporary Password: " + tempPass);

            // -------------------------------
            // 📜 PASSWORD HISTORY CHECK
            // -------------------------------
            boolean inHistory = pm.isPasswordInHistory(identity, "Rahul@123");
            System.out.println("Is in history: " + inHistory);

            String historyCheck = pm.checkPasswordHistory(identity, "Rahul@123");
            System.out.println("History check: " + historyCheck);

            List historyList = pm.getPasswordHistory(identity);
            System.out.println("Password History: " + historyList);

            // -------------------------------
            // 🎲 GENERATE PASSWORD
            // -------------------------------
            String randomPass = pm.generatePassword();
            System.out.println("Generated Password: " + randomPass);

            PasswordPolicy policy = pm.getPasswordPolicy(identity);
            String policyPass = pm.generatePassword(policy);
            System.out.println("Generated Policy Password: " + policyPass);

            // -------------------------------
            // 🚫 FAILED LOGIN / LOCK
            // -------------------------------
            pm.recordFailedLogin(identity);
            System.out.println("Failed login recorded");

            pm.resetFailedAttempts(identity);
            System.out.println("Failed attempts reset");

            boolean locked = pm.isAccountLocked(identity);
            System.out.println("Is Account Locked: " + locked);

            pm.unlockAccount(identity);
            System.out.println("Account unlocked");

            // -------------------------------
            // 📜 APPLY PASSWORD POLICY
            // -------------------------------
            String policyResult = pm.applyPasswordPolicy(identity, "Rahul@789");
            System.out.println("Policy Result: " + policyResult);

            // -------------------------------
            // 🔐 ENCRYPT / DECRYPT / HASH
            // -------------------------------
            String encrypted = pm.encryptPassword("Rahul@123");
            System.out.println("Encrypted: " + encrypted);

            String decrypted = pm.decryptPassword(encrypted);
            System.out.println("Decrypted: " + decrypted);

            String hashed = pm.hashPassword("Rahul@123");
            System.out.println("Hashed: " + hashed);

            // -------------------------------
            // 🔚 Close Context
            // -------------------------------
            context.close();

        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

🔹 checkPassword(Identity identity, String password) kya karta hai?
👉 Ye method (usually PasswordManager me) user ka diya hua password verify karta hai ki wo current stored password se match karta hai ya nahi.
🔥 Simple Hinglish me:-
👉 “Login jaisa check karta hai”
User ne password dala
System ne DB me stored (hashed) password se compare kiya
Match → ✅ true
Not match → ❌ false

🔹 Internally kya hota hai?
1️⃣ Identity ka stored password (hashed form me) uthata hai
2️⃣ Input password ko hash karta hai
3️⃣ Dono compare karta hai
4️⃣ Result return karta hai

🔹 Example Code (SailPointContext ke saath)
import sailpoint.api.SailPointContext;
import sailpoint.api.SailPointContextFactory;
import sailpoint.api.PasswordManager;
import sailpoint.object.Identity;

public class CheckPasswordDemo {

    public static void main(String[] args) {

        try {

            SailPointContextFactory factory = new SailPointContextFactory();
            SailPointContext context = factory.getCurrentContext();

            Identity identity = context.getObjectByName(Identity.class, "rahul.user");

            PasswordManager pm = new PasswordManager(context);

            boolean result = pm.checkPassword(identity, "Rahul@123");

            if(result) {
                System.out.println("Password correct ✅");
            } else {
                System.out.println("Password incorrect ❌");
            }

            context.close();

        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

🔹 validatePassword(Identity identity, String password) kya karta hai?
👉 Ye method (in PasswordManager):-
password ko complete policy ke against validate karta hai — sirf match nahi, balki full rules check karta hai.

🔥 🔹 Ye kya-kya validate karta hai?
1️⃣ Length Validation:-
✔ Minimum length
✔ Maximum length
👉 Example:-
Min = 8
Password = Rahul1 → ❌ fail

2️⃣ Character Rules:-
✔ Uppercase (A-Z)
✔ Lowercase (a-z)
✔ Numbers (0-9)
✔ Special Characters (@, #, $, etc.)
👉 Example:-
rahul123 → ❌ (no uppercase, no special char)

3️⃣ Character Classes:-
✔ Minimum different character types required
👉 Example:-
Required = 3 types
Password has only 2 → ❌ fail

4️⃣ Password Policy Rules:-
👉 Ye internally PasswordPolicy use karta hai
👉 PasswordPolicy
✔ Complexity rules
✔ Custom regex
✔ Rule-based validation

5️⃣ Password History Check:-
✔ Old password repeat nahi hona chahiye
👉 Example:-
Last passwords: Rahul@123, Rahul@456
New: Rahul@123 ❌

7️⃣ Custom Rule Validation:-
✔ Agar custom Java rule laga hai
✔ To wo bhi run hoga
👉 Example:-
Company rule: “No company name in password”

8️⃣ Expiration / Age Rules:-
✔ Password expired hai ya nahi
✔ Reuse allowed hai ya nahi

9️⃣ Dictionary / Weak Password Check (Optional):-
✔ Common passwords block karega
👉 Example:-
password123 ❌
