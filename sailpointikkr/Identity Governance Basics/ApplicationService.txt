()ApplicationService kya hai?
ApplicationService ek service layer component hota hai (especially SailPoint/backend systems me)
👉 iska kaam hota hai Application ko manage karna
Matlab:-
Application create karna

Fetch karna (DB se)

🔹 ApplicationService kya kya karta hai?
1️⃣ Application create karta hai:-
applicationService.createApplication(app);
👉 New application ko system me register karta hai

2️⃣ Application fetch karta hai:-
applicationService.getApplicationById(id);
👉 DB se application data laata hai

3️⃣ Application update karta hai:-
applicationService.updateApplication(app);
👉 Existing application ko modify karta hai

4️⃣ Application delete karta hai:-
applicationService.deleteApplication(id);
👉 Application remove kar deta hai

5️⃣ Aggregation / Sync trigger karta hai:-
👉 External system se data kheechta hai (accounts, roles, etc.)

import sailpoint.api.SailPointContext;
import sailpoint.api.SailPointFactory;
import sailpoint.object.Application;
import sailpoint.tools.xml.XMLObjectFactory;

public class ApplicationUtilityExample {

    public static void main(String[] args) {

        try {

            // 🔹 Get Context
            SailPointContext context = SailPointFactory.getCurrentContext();

            // 🔹 Get Application
            Application app = context.getObjectByName(Application.class, "SAP");

            if (app == null) {
                System.out.println("❌ Application not found");
                return;
            }

            // =====================================================
            // 🔥 1️⃣ VALIDATE APPLICATION
            // =====================================================
            System.out.println("\n--- VALIDATION START ---");

            if (app.getName() == null || app.getType() == null) {
                System.out.println("❌ Validation Failed: Name or Type missing");
            } else {
                System.out.println("✅ Basic fields valid");
            }

            if (app.getAccountSchema() == null) {
                System.out.println("❌ Validation Failed: Account Schema missing");
            } else {
                System.out.println("✅ Schema present");
            }

            if (app.getConnector() == null) {
                System.out.println("❌ Validation Failed: Connector missing");
            } else {
                System.out.println("✅ Connector configured");
            }

            System.out.println("--- VALIDATION END ---");


            // =====================================================
            // 🔥 2️⃣ CLONE APPLICATION
            // =====================================================
            System.out.println("\n--- CLONE APPLICATION ---");

            Application clone = (Application) app.deepCopy();
            clone.setName("SAP_COPY");

            context.saveObject(clone);
            context.commitTransaction();

            System.out.println("✅ Application cloned: " + clone.getName());


            // =====================================================
            // 🔥 3️⃣ EXPORT APPLICATION (XML)
            // =====================================================
            System.out.println("\n--- EXPORT APPLICATION ---");

            String xml = XMLObjectFactory.getInstance().toXml(app);
            System.out.println("📄 XML Output:\n" + xml);


            // =====================================================
            // 🔥 4️⃣ IMPORT APPLICATION
            // =====================================================
            System.out.println("\n--- IMPORT APPLICATION ---");

            String newXml = "<Application name='SAP_NEW' type='SAP'/>";
            Application importedApp = (Application) XMLObjectFactory.getInstance().parseXml(newXml);

            context.saveObject(importedApp);
            context.commitTransaction();

            System.out.println("✅ Application imported: " + importedApp.getName());


            // =====================================================
            // 🔥 5️⃣ REFRESH APPLICATION
            // =====================================================
            System.out.println("\n--- REFRESH APPLICATION ---");

            context.decache(app); // clear cache
            app = context.getObjectByName(Application.class, "SAP");

            System.out.println("✅ Application refreshed");


            // =====================================================
            // 🔥 6️⃣ COMPARE APPLICATIONS
            // =====================================================
            System.out.println("\n--- COMPARE APPLICATIONS ---");

            Application app1 = context.getObjectByName(Application.class, "SAP");
            Application app2 = context.getObjectByName(Application.class, "SAP_COPY");

            if (app1 != null && app2 != null) {
                if (app1.getType().equals(app2.getType())) {
                    System.out.println("✅ Same Application Type");
                } else {
                    System.out.println("❌ Different Types");
                }
            }


            // =====================================================
            // 🔥 7️⃣ DELETE APPLICATION (OPTIONAL ⚠️)
            // =====================================================
            System.out.println("\n--- DELETE APPLICATION ---");

            Application deleteApp = context.getObjectByName(Application.class, "SAP_NEW");

            if (deleteApp != null) {
                context.removeObject(deleteApp);
                context.commitTransaction();
                System.out.println("✅ Application deleted: SAP_NEW");
            } else {
                System.out.println("ℹ️ No application found to delete");
            }

        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

👉 Jo ek XMLObjectFactory ka single shared object (instance) return karta hai,baar baar object banane se chutkara deta hai.

🔹 Internal concept (samajhne ke liye):-
public class XMLObjectFactory {

    private static XMLObjectFactory instance;

    private XMLObjectFactory() {
        // private constructor
    }

    public static XMLObjectFactory getInstance() {
        if (instance == null) {
            instance = new XMLObjectFactory();
        }
        return instance;
    }
}
