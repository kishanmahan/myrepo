()Normalization kya hota hai?
Normalization ek process hai jisme hum data ko proper structure me organize karte hain, taki:
Data duplicate na ho
Data clean aur consistent rahe
Database efficient ho jaaye

🔥 Easy Example:-
❌ Without Normalization (Galat way)
Student	Course	Teacher
Rahul	Java	Amit
Rahul	Python	Amit

👉 Problem:-
Rahul ka naam baar-baar repeat ho raha hai
Teacher bhi duplicate ho raha hai
✅ After Normalization (Sahi way)

Table 1: Students:-
ID	    Name
1	    Rahul

Table 2: Courses:-
ID	Course
1	Java
2	Python
------------------------------------------------------
🔷 SailPoint me Normalization kya hai?
Normalization ka matlab hota hai:-
👉 Alag-alag systems se aane wale data ko ek common format me convert karna

🧠 Simple Language:-
Different applications (AD, SAP, DB, etc.) me data alag format me hota hai:

System	Username Field
AD	   sAMAccountName
SAP	    user_id
DB	    login_name

👉 SailPoint kya karta hai?
➡️ In sabko ek hi field me map karta hai (jaise: identityName)
👉 Is process ko hi Normalization bolte hain.
