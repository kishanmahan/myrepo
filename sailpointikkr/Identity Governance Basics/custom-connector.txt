()Connector kya hai?
--Connector = SailPoint aur Target System ke beech ka bridge (connection)
Matlab SailPoint directly system se baat nahi karta.
Wo connector use karta hai taaki target system se data le sake aur access manage kar sake.

Simple Example:-
Company me employee Rahul hai.
Uska account Active Directory me banana hai.
Flow:-

Rahul
↓
SailPoint
↓
Connector
↓
Active Directory
↓
Account Create

Yaha Connector SailPoint ko Active Directory se connect kar raha hai.
