🔐 IAM Kya Hota Hai?
IAM = Identity and Access Management
Ye ek security framework hai jo manage karta hai:-
👤 User authentication (login kaise karega)
🔑 Authorization (kya access milega)
🛡️ Access control
🔄 Account lifecycle
📊 Security monitoring

🆚 IAM vs IGA (Short Difference):-
IAM	                                 IGA
Access dena-------------------Access review karna
Authentication-------------------Governance
Real-time login control-------Compliance & Audit
Operational focus-------------Regulatory focus
Example:-
IAM → Login allow karega
IGA → Check karega access sahi hai ya nahi
