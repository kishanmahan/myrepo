1️⃣1️⃣ Identity Lifecycle Management:-
Lifecycle stages:-
Joiner
Mover
Leaver

6️⃣ Identity Lifecycle kya hai?
Ye user ka complete lifecycle manage karta hai.

Stages:-
1️⃣ Joiner:-
New employee join karta hai

2️⃣ Mover:-
Employee department change karta hai

3️⃣ Leaver:-
Employee company leave karta hai

Example:-
Joiner:-
➡ Account create

Mover:-
➡ Old access remove + new access add

Leaver:-
➡ All access remove
----------------------------------------
1️⃣ Identity Lifecycle Overview
ILM automate करता है:-
✔ Account creation
✔ Role assignment
✔ Access modification
✔ Account disable / delete
Flow:-
HR System
   ↓
Identity Created
   ↓
Access Assigned
   ↓
Role Change
   ↓
Access Updated
   ↓
User Exit
   ↓
Access Removed

2️⃣ Joiner Process (New Employee):-
जब नया employee join करता है।
Steps:-
1️⃣ HR system में user create होता है
2️⃣ SailPoint aggregation करता है
3️⃣ Identity create होती है
4️⃣ Birthright roles assign होते हैं
5️⃣ Accounts automatically create होते हैं

Example:-
User : Rahul
Department : Finance

System automatically assign करेगा:-
Finance Role
Finance Application Access
Email Account
VPN Access

3️⃣ Joiner Configuration Methods:-
1️⃣ Birthright Roles:-
Automatically roles assign होते हैं।
Example:-
Department = HR
Role = HR_User

2️⃣ Lifecycle Events:-
Events trigger करते हैं automation।
Example:-
New Identity Created

Trigger:-
Create accounts
Assign roles

3️⃣ Provisioning Plan:-
Account creation के लिए provisioning plan बनता है।

Example:-
Operation: Create
Application: Active Directory

4️⃣ Mover Process (Role Change):-
Mover तब होता है जब user:-
✔ Department change करता है
✔ Promotion होता है
✔ Role change होता है

Example:-
Old Role: HR_User
New Role: HR_Manager

System करेगा:-
Remove HR_User entitlements
Add HR_Manager entitlements

5️⃣ Mover Automation Methods:-
1️⃣ Role Change Detection
System detect करता है:
Department change
Location change
Manager change

2️⃣ Recalculation:-
Identity refresh होता है।
Identity Refresh Task
यह check करता है:-
Old roles remove
New roles add

6️⃣ Leaver Process (Employee Exit):-
जब employee company छोड़ता है।
System automatically:-
✔ Accounts disable करता है
✔ Access remove करता है
✔ Roles revoke करता है

Example:-
Employee status = Terminated

Action:-
Disable accounts
Remove roles
Revoke entitlements

7️⃣ Leaver Automation Methods:-
1️⃣ Deprovisioning:-
Access remove होता है।
Example:-
Disable AD account
Disable Email
Remove VPN

2️⃣ Account Deletion:-
कुछ systems में account delete भी होता है।
Delete account operation

8️⃣ Lifecycle Events:-
Lifecycle automation events से trigger होता है।
Examples:-
Joiner event
Mover event
Leaver event

Example configuration:-
Event: Identity Created
Action: Assign Birthright Role

9️⃣ Lifecycle States:-
