3. JDBC Provisioning Rule (Asli Magic):-
Kyunki ye JDBC hai, SailPoint ko ye nahi pata ki table mein data kaise daalna hai. 
Humein ek chota sa Beanshell Rule likhna hoga jo INSERT query chalaye.
Application mein Rules tab par jayein.
JDBC Provisioning Rule wale dropdown mein Create New Rule karein (Ya agar pehle se bana hai toh wahi use karein).
Usme ye basic logic daalna hoga:-

()Provision Rule:-Is dropdown ke bagal mein jo ... (Three dots) wala button hai, us par click karein. 
Ek naya popup khulega jisme hum Beanshell code likhenge.

import java.sql.Connection;
import java.sql.PreparedStatement;
import sailpoint.object.ProvisioningPlan;
import sailpoint.object.ProvisioningPlan.AccountRequest;
import sailpoint.object.ProvisioningPlan.AttributeRequest;
import sailpoint.api.Provisioner;

// 1. Connection check karein (SailPoint automatically 'connection' object deta hai)
if (connection == null) {
    return; 
}

// 2. Plan se Account Requests nikalna
List acctReqs = plan.getAccountRequests();
if (acctReqs != null) {
    for (AccountRequest acctReq : acctReqs) {
        // Sirf 'Create' operation ke liye logic
        if (AccountRequest.Operation.Create.equals(acctReq.getOperation())) {
            
            // Attributes nikalna
            String v_user = (String) acctReq.getAttributeValue("username");
            String v_first = (String) acctReq.getAttributeValue("firstname");
            String v_last = (String) acctReq.getAttributeValue("lastname");
            String v_email = (String) acctReq.getAttributeValue("email");

            // MySQL INSERT Query
            String sql = "INSERT INTO target_users (username, firstname, lastname, email) VALUES (?, ?, ?, ?)";
            
            PreparedStatement st = connection.prepareStatement(sql);
            st.setString(1, v_user);
            st.setString(2, v_first);
            st.setString(3, v_last);
            st.setString(4, v_email);
            
            st.executeUpdate();
            st.close();
        }
    }
}
