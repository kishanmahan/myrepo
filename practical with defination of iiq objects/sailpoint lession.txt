​3. SailPoint Ke Andar Custom REST API Banana:-
​Agar SailPoint ki pehle se bani-banayi (OOTB) APIs aapke kaam nahi aa rahi hain aur aapko ek custom logic chahiye, toh aap SailPoint ke andar khud ki REST API bhi likh sakte hain:
​SailPoint Java par chalta hai, isliye aap iske andar Custom Resource classes (JAX-RS / Jersey annotations jaise @Path, @GET, @POST ka use karke) likhte hain.
​Is custom Java class ko aap SailPoint ke backend plugin structure mein deploy kar dete hain. Iske baad aapka SailPoint ek poore web server ki tarah custom API endpoints handle karne lagta hai.
