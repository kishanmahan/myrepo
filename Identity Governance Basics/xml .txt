<TaskDefinition name="Demo Provisioning Task" type="Generic">
    <Description>Demo Task for Provisioning</Description>
    
    <Attributes>
        <Map>
            <entry key="executor" value="com.company.task.DemoTaskExecutor"/>
        </Map>
    </Attributes>

🔹 <TaskDefinition ... >
👉 Ye main root tag hai
👉 SailPoint ko bolta hai:
"Ye ek task define kiya ja raha hai"

🔹 name="Demo Provisioning Task":-
👉 Task ka display name (UI me dikhega)
Admin UI → Tasks → Demo Provisioning Task
🔹 type="Generic"
👉 Task ka type define karta hai

Type	                     Meaning
Generic:---------------Custom task (developer ka code run karega)
AccountAggregation:----Aggregation task
IdentityRefresh:-------Identity update task
Generic = Custom Executor run hoga,custom(generic) task ke liye.

🔹 <entry key="executor" value="..."/>:-
👉 MOST IMPORTANT LINE 🔥
key = executor

👉 Meaning:-
"Task engine, is class ko run karo"

👉 Yaha:-
com.company.task.DemoTaskExecutor

👉 Matlab:-
➡️ Ye Java class load hogi
➡️ iska execute() method call hoga.
