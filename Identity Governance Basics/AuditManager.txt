()🔥 AuditManager kya hai?
AuditManager bhi SailPoint IdentityIQ me ek class/service hoti hai
Ye AuditEvent ko create, log, save aur manage karne ke liye use hoti hai

👉 Simple:-
AuditEvent = data (log record)
AuditManager = us data ko handle karne wala controller/service.

🔹 AuditManager ke methods:-
log(AuditEvent event)
log(String action, String actor, String target)
log(String action, String actor, String target, String result)
log(AuditEvent event, boolean persist)
save(AuditEvent event)

🔥 1️⃣ log(AuditEvent event):-
📌 Purpose:-
Ek complete AuditEvent ko log + persist (DB me save) karta hai
⚙️ Internally:-
Event validate hota hai
Timestamp set hota hai
DB me insert hota hai
🧪 Example:-
AuditEvent event = new AuditEvent();
event.setAction("LOGIN");
event.setActor("User1");
auditManager.log(event);
✅ Output:-
Audit Event Logged
