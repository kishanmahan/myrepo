👉 RiskScore = ek numeric value (number)
jo batata hai ki user / role / entitlement kitna risky hai
🧠 Simple example:-
User A → RiskScore = 20  (safe)
User B → RiskScore = 80  (high risk 🚨)
👉 Matlab:-
Jitna zyada score → utna zyada security risk

🎯 RiskScore kya karta hai?
👉 Ye help karta hai:-
🔐 High-risk users identify karna
🚨 Sensitive access track karna
📊 Audit & compliance me Policy violation detect karna.
------------------------------------------------------------------------------------------------
🧱 RiskScore ke andar kya hota hai?
RiskScore normally direct value nahi hota, ye calculate hota hai multiple cheezon se 👇

🔹 1. Entitlements Risk:-
👉 Agar user ke paas dangerous access hai:-
Admin access → high risk
Normal user → low risk

🔹 2. Role Risk:-
👉 Role ka bhi apna risk hota hai:-
Role: Database Admin → Risk = 90
Role: Viewer → Risk = 10

🔹 3. Application Risk:-
👉 Kuch apps sensitive hote hai:-
SAP → high risk
Email → medium risk

🔹 4. Policy Violations
👉 Agar user rule tod raha hai:-
SoD violation → risk increase

🔹 5. Weight (Importance):-
👉 Har cheez ka weight hota hai:-
Entitlement weight = 50%
Role weight = 30%
Policy = 20%
👉 Final RiskScore calculate hota hai.
--------------------------------------------------
⚙️ Internally kaise banta hai?
RiskScore = (Entitlements Risk × weight)
          + (Role Risk × weight)
          + (Policy Risk × weight)
------------------------------------------------------
💻 Example (Real Feel):-
User ke paas:-
Admin entitlement → 80
Role → 50
Policy violation → 20
Final RiskScore ≈ 65

import sailpoint.object.Identity;
import sailpoint.object.Bundle;
import sailpoint.object.ManagedAttribute;

public class RiskUtil {

    public static int calculateRiskScore(Identity identity) {

        int totalRisk = 0;

        // 🔹 Entitlements risk
        java.util.List<ManagedAttribute> ents = identity.getEntitlements();
        for (ManagedAttribute ent : ents) {
            totalRisk += ent.getRiskScore();
        }

        // 🔹 Roles risk with weight
        java.util.List<Bundle> roles = identity.getRoles();
        for (Bundle role : roles) {
            int roleRisk = role.getRiskScore();
            int weight = role.getRiskScoreWeight();

            totalRisk += (roleRisk * weight) / 100;
        }

        // 🔹 Policy violations
        java.util.List violations = identity.getPolicyViolations();
        totalRisk += violations.size() * 10;

        return totalRisk;
    }
}
✅ Output:-

System.out.println(risk);

import sailpoint.api.SailPointContext;
import sailpoint.object.Identity;
import sailpoint.tools.GeneralException;

public class RiskUpdater {

    public static void updateRiskScore(SailPointContext context, Identity identity) throws GeneralException {

        int risk = RiskUtil.calculateRiskScore(identity);

        // 🔹 set risk
        identity.setRiskScore(risk);

        // 🔹 save in DB
        context.saveObject(identity);
        context.commitTransaction();

        System.out.println("RiskScore updated: " + risk);
    }
}

Risk matlab uncertainty jo loss ya problem la sakti hai.
Example:-
Company ka server crash ho sakta hai → data loss ka risk.

2️⃣ Risk Management ka process:-
Risk management generally 4 main steps me hota hai:-

()Risk Identification (Risk pehchanna):-
Sabse pehle ye check karte hain ki kaunse areas me problems ya losses ho sakte hain.
Example: IT system, finance, human resources, projects

()Risk Analysis (Risk ka analysis karna):-
Risk ke impact aur probability ko samjhte hain
Example: Agar server crash hua → company ka loss ₹1 crore ho sakta hai

()Risk Evaluation (Risk ko rank karna):-
Kaunsa risk sabse zyada dangerous hai, kaunsa kam dangerous
High, Medium, Low risk me categorize karte hain

()Risk Treatment / Mitigation (Risk ko control karna):-
Risk ko avoid, reduce, share, ya accept karna
Example: Backup servers lagana → data loss ka risk kam ho jata hai.

()Risk scoring kya hai?
--1️⃣ Risk Scoring ka matlab:-
Risk ko quantify karna (number me convert karna)
Ye score risk ke probability × impact se nikalta hai
Formula:-
Risk Score=Probability of Risk×Impact of Risk
Risk Score=Probability of Risk×Impact of Risk

Example:-
Server crash hone ka probability = 0.2 (20%)
Impact = ₹1,00,000
Risk Score = 0.2 × 1,00,000 = ₹20,000
Matlab ye risk moderate hai, zyada nahi.

////////*
probability basically ye batati hai ki koi particular risk kitni chances hai ki actually ho jaye.
Medium Probability → Risk ka chance moderate.
Data Breach ka matlab hai ki sensitive ya confidential data unauthorized logon ke haath lag jana. Ye company, users, ya organizations ke liye kaafi bada risk hai.

-Risk identify karna:-

-Probability assign karna:-

import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

// Risk class
class Risk {
    private String name;
    private double probability; // 0.0 to 1.0
    private double impact;      // in currency
    private double score;
    private String priority;

    // Constructor
    public Risk(String name, double probability, double impact) {
        this.name = name;
        this.probability = probability;
        this.impact = impact;
        calculateScore();
        assignPriority();
    }

    // Score calculation
    private void calculateScore() {
        this.score = this.probability * this.impact;
    }

    // Priority assignment
    private void assignPriority() {
        if (score >= 75000) {
            this.priority = "High";
        } else if (score >= 25000) {
            this.priority = "Medium";
        } else {
            this.priority = "Low";
        }
    }

    // Display Risk
    public void displayRisk() {
        System.out.println("Risk Name   : " + name);
        System.out.println("Probability : " + probability);
        System.out.println("Impact      : " + impact);
        System.out.println("Risk Score  : " + score);
        System.out.println("Priority    : " + priority);
        System.out.println("--------------------------");
    }
}

// Main class
public class RiskManagementProject {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        List<Risk> riskList = new ArrayList<>();

        System.out.println("=== Risk Management & Scoring Project ===");

        while (true) {
            System.out.println("\n1. Add Risk");
            System.out.println("2. Show All Risks");
            System.out.println("3. Exit");
            System.out.print("Choose option: ");
            int choice = sc.nextInt();
            sc.nextLine(); // Consume newline

            if (choice == 1) {
                System.out.print("Enter Risk Name: ");
                String name = sc.nextLine();

                System.out.print("Enter Probability (0.0 - 1.0): ");
                double prob = sc.nextDouble();

                System.out.print("Enter Impact (₹): ");
                double impact = sc.nextDouble();

                Risk r = new Risk(name, prob, impact);
                riskList.add(r);
                System.out.println("✅ Risk added successfully!");

            } else if (choice == 2) {
                if (riskList.isEmpty()) {
                    System.out.println("No risks added yet.");
                } else {
                    System.out.println("\n--- All Risks ---");
                    for (Risk r : riskList) {
                        r.displayRisk();
                    }
                }
            } else if (choice == 3) {
                System.out.println("Exiting... Bye!");
                break;
            } else {
                System.out.println("Invalid choice! Try again.");
            }
        }

        sc.close();
    }
}
