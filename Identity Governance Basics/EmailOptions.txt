()EmailOptions = email bhejne ke time extra settings aur data dene ke liye object
Isme hum decide karte hain:-
email kis ko jayega
CC / BCC
variables
attachments
reply address

1️⃣ Simple Example:-
import sailpoint.object.EmailTemplate;
import sailpoint.object.EmailOptions;

EmailTemplate template = context.getObjectByName(
        EmailTemplate.class, "Access Request Approved");

EmailOptions options = new EmailOptions("rahul@gmail.com", null);

context.sendEmailNotification(template, options);
Output;-
Email sent to rahul@gmail.com

2️⃣ EmailOptions Constructors:-
1️⃣ EmailOptions(String to, Map vars):-
EmailOptions options = new EmailOptions("rahul@gmail.com", null);
to → receiver email
vars → template variables

Kya karta hai?
Email kis user ko bhejna hai ye set karta hai.
Code:-
EmailOptions options = new EmailOptions("rahul@gmail.com", null);
options.setTo("rahul@gmail.com");

System.out.println(options.getTo());
Output
rahul@gmail.com

🔟 setVariables(Map vars)
Kya karta hai

EmailTemplate ke dynamic variables set karta hai.

Example template:

Hello ${identityName}
Application: ${applicationName}

(1)Code:-
import java.util.HashMap;
import java.util.Map;
import sailpoint.object.EmailOptions;

Map vars = new HashMap();

vars.put("identityName","Rahul");
vars.put("applicationName","SAP");

EmailOptions options = new EmailOptions("rahul@gmail.com", null);

options.setVariables(vars);
System.out.println(options.getVariables());

Output:-
{identityName=Rahul, applicationName=SAP}

Purpose:-
Template ke placeholders ko value dena

1️⃣2️⃣ setAttachments(List attachments)
Kya karta hai

Email ke saath files attach karta hai.

Example:

PDF
Excel
Logs
Reports
Code
import java.util.ArrayList;
import java.util.List;

List files = new ArrayList();

files.add("report.pdf");
files.add("access.xlsx");

EmailOptions options = new EmailOptions("rahul@gmail.com", null);

options.setAttachments(files);

System.out.println(options.getAttachments());
Output
[report.pdf, access.xlsx]

Purpose:-
Email ke saath documents bhejna
