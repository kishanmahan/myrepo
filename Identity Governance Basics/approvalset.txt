()ApprovalSet SailPoint me ek object hai jo workflow me approvals ko manage karne ke liye use hota hai.
Simple shabdon me, ye batata hai ki kaun approve kar raha hai, kaun pending hai aur kaun reject kar chuka hai.

()approvalSet me kya hota hai?
ApprovalSet ek object hota hai jisme:-
approval items
entitlement changes
account changes
approve/reject decisions
store hote hain.

()ApprovalSet ka kaam:-
1️⃣ Approvers list rakhna – Kaun log approve karenge ya kar chuke hain.
2️⃣ Pending approvals track karna – Kaun abhi action nahi liya.
3️⃣ Approval status maintain karna – Approved, Pending, Rejected.
4️⃣ Workflow me decisions control karna – Agar saare approvals mil gaye → workflow aage badhe.

Example Code:-
ApprovalSet set = new ApprovalSet();
set.addApprover("manager1"); // Approver add kiya
set.addApprover("manager2");
workflowCase.setApprovalSet(set);
System.out.println(workflowCase.getApprovalSet().getApprovers());

Output:-
[manager1, manager2]

()ApprovalSet – Approval Status Types:-
1️⃣ Pending:-
Approver ne abhi approve/reject action nahi liya.

2️⃣ Approved:-
Approver ne task ya request approve kar diya.

3️⃣ Rejected:-
Approver ne task ya request reject kar diya.

4️⃣ Escalated (optional, implementation dependent):-
Agar approver ne action nahi liya aur escalation ho gayi, status Escalated ho sakta hai.

5️⃣ Not Required (optional, implementation dependent):-
Kabhi kabhi approval ki requirement cancel ho jaye → status Not Required.

()✅ validate() kya-kya check karta hai:-
1️⃣ Approvers empty na ho:-
ApprovalSet me kam se kam 1 approver hona chahiye

set = new ApprovalSet();
set.validate();

❌ Output:-
ValidationException: No approvers defined

2️⃣ Duplicate approvers na ho:-
Same approver 2 baar nahi hona chahiye

set.addApprover("manager1");
set.addApprover("manager1");
set.validate();

❌ Output:-
ValidationException: Duplicate approver

3️⃣ Valid approval status:-
Status sirf valid hona chahiye:-
Pending
Approved
Rejected

set.setApprovalStatus("manager1","DONE");
set.validate();

❌ Output:-
ValidationException: Invalid status

4️⃣ Har approver ka status hona chahiye:-
Har approver ke paas ek valid status assign hona chahiye

set.addApprover("manager2");
set.validate();

❌ Output:-
ValidationException: Status missing for approver

5️⃣ Null values check:-
Approver name ya Identity null nahi hona chahiye

set.addApprover((String) null);
set.validate();

❌ Output:-
ValidationException: Approver cannot be null

6️⃣ Consistency check:-
Example:-
Sab Approved → valid
Mixed → valid
Invalid combination → error

✅ 1️⃣ Sab Approved → VALID:-
Sab approvers ne approve kar diya

ApprovalSet set = new ApprovalSet();
set.addApprovers(List.of("m1","m2"));

set.setApprovalStatus("m1","Approved");
set.setApprovalStatus("m2","Approved");

set.validate();
System.out.println("Valid");

Output:-
Valid
✔️ Reason: Sab approve → workflow aage badh sakta hai

✅ 2️⃣ Mixed (Approved + Pending) → VALID:-
Kuch approve, kuch pending
set.setApprovalStatus("m1","Approved");
set.setApprovalStatus("m2","Pending");
set.validate();
System.out.println("Valid");
Output:-
Valid
✔️ Reason: Abhi process chal raha hai → allowed hai

❌ 3️⃣ Invalid Combination → ERROR:-
Case 1: Rejected + Approved (conflict case)
set.setApprovalStatus("m1","Approved");
set.setApprovalStatus("m2","Rejected");
set.validate();
Output:-
ValidationException: Conflicting approval states

👉 Reason:-
Ek ne approve kiya
Ek ne reject kiya
➡️ Final decision clear nahi → conflict

Case 2: Final state + Pending (invalid flow)
set.setApprovalStatus("m1","Rejected");
set.setApprovalStatus("m2","Pending");
set.validate();
Output:-
ValidationException: Invalid state combination

👉 Reason:-
Reject already ho gaya → process end hona chahiye
Pending still hai → inconsistent

✅ ApprovalSet Methods – Real Project Code:-
import sailpoint.api.SailPointContext;
import sailpoint.object.ApprovalSet;
import sailpoint.object.Identity;

import java.util.Arrays;
import java.util.List;
import java.util.Map;

public class ApprovalSetExample {

    public static void main(String[] args) {

        SailPointContext context = null;

        try {
            // 🔹 Get SailPoint Context
            context = SailPointFactory.createContext();

            // 🔹 Create ApprovalSet object
            ApprovalSet approvalSet = new ApprovalSet();

            // ============================================
            // 1️⃣ Add Approvers
            // ============================================

            // Add single approver (String)
            approvalSet.addApprover("john.doe");

            // Add single approver (Identity)
            Identity id1 = context.getObjectByName(Identity.class, "jane.smith");
            approvalSet.addApprover(id1);

            // Add multiple approvers (String list)
            List<String> approverNames = Arrays.asList("user1", "user2");
            approvalSet.addApprovers(approverNames);

            // Add multiple approvers (Identity list)
            Identity id2 = context.getObjectByName(Identity.class, "manager1");
            Identity id3 = context.getObjectByName(Identity.class, "manager2");
            approvalSet.addApprovers(Arrays.asList(id2, id3));

            // ============================================
            // 2️⃣ Remove Approvers
            // ============================================

            approvalSet.removeApprover("user1");

            Identity removeId = context.getObjectByName(Identity.class, "manager2");
            approvalSet.removeApprover(removeId);

            // ============================================
            // 3️⃣ Validate ApprovalSet
            // ============================================

            boolean isValid = approvalSet.validate();
            System.out.println("Is ApprovalSet Valid? " + isValid);

            // ============================================
            // 4️⃣ Convert to XML
            // ============================================

            String xml = approvalSet.toXml();
            System.out.println("ApprovalSet XML:");
            System.out.println(xml);

            // ============================================
            // 5️⃣ Convert from XML
            // ============================================

            ApprovalSet newApprovalSet = new ApprovalSet();
            newApprovalSet.fromXml(xml);
            System.out.println("ApprovalSet recreated from XML");

            // ============================================
            // 6️⃣ Convert to Map
            // ============================================

            Map map = approvalSet.toMap();
            System.out.println("ApprovalSet Map: " + map);

            // ============================================
            // 7️⃣ Convert from Map
            // ============================================

            ApprovalSet mapApprovalSet = new ApprovalSet();
            mapApprovalSet.fromMap(map);
            System.out.println("ApprovalSet recreated from Map");

            context.commitTransaction();

        } catch (Exception e) {
            e.printStackTrace();

        } finally {
            if (context != null) {
                context.decache(true);
                context.close();
            }
        }
    }
}
