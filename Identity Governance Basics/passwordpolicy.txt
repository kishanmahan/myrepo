1️⃣ Password Policy क्या होती है
Password Policy ऐसी security policy होती है जो password की strength, length और complexity को enforce करती है।
मतलब password में यह rules होने चाहिए:-
Minimum length
Uppercase letter
Lowercase letter
Number
Special character
अगर rule follow नहीं हुआ → Password Policy Violation.

3️⃣ Project Structure
password-policy-project
│
├── PasswordPolicyEngine.java
└── PasswordPolicyMain.java
4️⃣ Password Policy Engine (Enterprise Style)
public class PasswordPolicyEngine {

    public static void validatePassword(String password) {

        boolean hasUpper = false;
        boolean hasLower = false;
        boolean hasDigit = false;
        boolean hasSpecial = false;

        if(password.length() < 8) {
            System.out.println("Policy Violation: Password length must be at least 8");
            return;
        }

        for(char c : password.toCharArray()) {

            if(Character.isUpperCase(c))
                hasUpper = true;

            else if(Character.isLowerCase(c))
                hasLower = true;

            else if(Character.isDigit(c))
                hasDigit = true;

            else
                hasSpecial = true;
        }

        if(!hasUpper) {
            System.out.println("Policy Violation: Missing uppercase letter");
            return;
        }

        if(!hasLower) {
            System.out.println("Policy Violation: Missing lowercase letter");
            return;
        }

        if(!hasDigit) {
            System.out.println("Policy Violation: Missing number");
            return;
        }

        if(!hasSpecial) {
            System.out.println("Policy Violation: Missing special character");
            return;
        }

        System.out.println("Password Accepted (Strong Password)");

    }
}
5️⃣ Main Program
public class PasswordPolicyMain {

    public static void main(String[] args) {

        String password1 = "password";
        String password2 = "Password1";
        String password3 = "Password@1";

        PasswordPolicyEngine.validatePassword(password1);

        System.out.println("-------------------");

        PasswordPolicyEngine.validatePassword(password2);

        System.out.println("-------------------");

        PasswordPolicyEngine.validatePassword(password3);
    }
}
